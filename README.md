# Mini-Projeto-Curricular-Senac-Analise-de-Dados
Mini projeto referente à conclusão de unidade curricular do curso de Analista de Dados Senac. Nele foi abordado o ciclo da Análise de Dados mediante arquivo ipynb.


Mini Projeto – Análise de Dados em Saúde
Visão Geral

Este repositório contém um mini projeto de Análise de Dados desenvolvido em Python como atividade acadêmica (SENAC-CE). O objetivo do projeto é realizar a importação, exploração, limpeza, tratamento e visualização de dados a partir de um conjunto de dados da área da saúde (healthcare dataset), aplicando conceitos fundamentais de Ciência de Dados.
O projeto foi desenvolvido em um Jupyter Notebook, permitindo a combinação de código, visualizações gráficas e explicações conceituais.

Autor:
Aluno: Antonio Marcos Feitosa da Silva

Objetivos do Projeto:

Importar e explorar um conjunto de dados da área da saúde
Compreender a estrutura e os tipos de dados disponíveis
Realizar limpeza e padronização das informações
Tratar valores ausentes e inconsistências
Gerar análises estatísticas descritivas
Criar visualizações gráficas para apoiar a análise dos dados

Etapas Desenvolvidas:

1. Entendimento do Projeto e Importação dos Dados:

Importação das bibliotecas necessárias
Leitura do arquivo CSV contendo os dados de saúde
Visualização inicial do dataset
Análise da estrutura dos dados (tipos de colunas, quantidade de registros)

2. Exploração Inicial dos Dados:

Uso de métodos como:
head()
info()
describe()
Identificação de padrões, valores ausentes e possíveis inconsistências

3. Limpeza e Preparação dos Dados:

Padronização dos nomes das colunas
Conversão de tipos de dados
Tratamento de valores nulos
Ajustes para facilitar análises posteriores

4. Análise dos Dados:

Análises estatísticas básicas
Agrupamentos e contagens relevantes
Exploração de variáveis categóricas e numéricas

5. Visualização de Dados:

Criação de gráficos utilizando Matplotlib
Visualizações para melhor compreensão dos dados analisados

Tecnologias Utilizadas:
Python 3
Jupyter Notebook
Pandas – Manipulação e análise de dados
NumPy – Operações numéricas
Matplotlib – Visualização de dados
Google Colab
